== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `remove_utf8_permalink/remove_utf8_permalink.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
Plugin will automatic runing when you post new entry.
